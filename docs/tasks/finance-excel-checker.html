Module 01

Finance Excel Checker

Практичне завдання: Git + GitHub + Codex CLI + Excel/CSV для фінансового відділу.

Мета

Навчитися створювати локальний Git-проєкт, працювати з GitHub, використовувати окрему гілку для змін і застосовувати Codex CLI для створення простого інструмента перевірки фінансових даних.

Що треба зробити

  1. Створити локальний проєкт finance-excel-checker.
  2. Додати файл payments.csv з фінансовими даними.
  3. Зробити перший commit.
  4. Створити окрему Git-гілку.
  5. Запустити Codex CLI.
  6. Попросити Codex створити Python-скрипт перевірки платежів.
  7. Перевірити результат.
  8. Оформити README.
  9. Запушити проєкт на GitHub.

Структура проєкту студента

finance-excel-checker/
├── payments.csv
├── check_payments.py
├── README.md
└── .gitignore

Крок 1. Створити проєкт

mkdir finance-excel-checker
cd finance-excel-checker
git init
git branch -M main

Крок 2. Додати CSV-файл

Створи файл payments.csv або завантаж приклад: payments.csv

Крок 3. Перший commit

git add .
git commit -m "Add sample payments data"

Крок 4. Створити окрему гілку

git switch -c add-payment-validation

Важливо: зміни через Codex CLI краще робити не в main, а в окремій гілці.

Крок 5. Запустити Codex CLI

codex

Спочатку попроси Codex пояснити проєкт:

Can you give me a brief summary of this project?

Крок 6. Завдання для Codex

Скопіюй цей prompt у Codex CLI:

Create a simple Python script called check_payments.py that reads payments.csv and checks for common finance data issues: missing invoice numbers, negative amounts, zero amounts, and possible duplicate payments. The script should print a clear report in the terminal.

Крок 7. Запустити перевірку

python3 check_payments.py

Крок 8. Перевірити зміни Git

git status
git diff

Крок 9. Commit результату

git add .
git commit -m "Add payment validation script"

Крок 10. README

Попроси Codex створити README:

Create a README.md file for this project. Explain what the project does, what files it contains, and how to run the payment validation script.

Після цього:

git add README.md
git commit -m "Add project README"

Що треба здати

  • посилання на GitHub-репозиторій;
  • файл payments.csv;
  • файл check_payments.py;
  • файл README.md;
  • історію мінімум з трьох commit;
  • скріншот запуску python3 check_payments.py.
